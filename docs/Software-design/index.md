## 设计的层次

设计是一个非常宽泛的概念，对于一个系统，从整体到局部、从顶层到底层，我们都需要进行详细的设计，这说明对于一个完整的系统，我们需要从各个层次对它进行设计。关于设计的层次，在`Design-pattern\index.md`中进行了较好的总结。
